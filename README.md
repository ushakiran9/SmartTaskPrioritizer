# SmartTaskPrioritizer

A feature-rich desktop productivity manager built using **Java Swing**. SmartTaskPrioritizer empowers users to organize, prioritize,
 and track daily tasks with an intuitive graphical interface and advanced productivity features.

## Features

- **Add, Edit, and Delete Tasks:** Easily manage your to-do items through interactive forms.
- **Smart Priority Scoring:** Automatically ranks tasks using importance, effort, and due date.
- **Color-Coded Urgency:** Visual cues highlight urgent and overdue tasks.
- **Advanced Search and Filtering:** Quickly locate tasks by keyword or category.
- **CSV Export:** Download your task list for backup or analysis.
- **Persistent Storage:** All tasks are saved and automatically reloaded on next use.
- **Modern, Accessible GUI:** Large fonts, bold controls, and a professional, user-friendly layout.



## How to Run

1. **Requirements**
   - Java JDK 8 or newer (tested up to Java 21)
   - OS: Windows, Linux

2. **Compile and Run**
   ```
   javac SmartTaskPrioritizer.java
   java SmartTaskPrioritizer
   ```

3. **Usage**
   - Follow the intuitive interface to add, edit, complete, search, or export your daily tasks.
